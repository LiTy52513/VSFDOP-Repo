# Data Availability

This repository contains a self-contained in-memory example and does not require external data files to build or run the core VSF-DOP implementation.

The benchmark cases in the associated study are generated from the reported geometric parameters. Large generated benchmark meshes and performance-output files are not included in this minimal source package.

Suggested data availability statement:

> The code package includes a self-contained example for verifying the VSF-DOP implementation. The benchmark geometries used in the associated study are generated from the reported case definitions.
