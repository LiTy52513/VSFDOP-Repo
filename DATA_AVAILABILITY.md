# Data Availability

This repository contains a self-contained in-memory example and does not require external data files to build or run the core VSF-DOP implementation.

The manuscript benchmark cases are generated from the geometric parameters described in the paper. Large generated benchmark meshes and performance-output files are not included in this minimal source package.

Suggested manuscript statement:

> The code package includes a self-contained example for verifying the VSF-DOP implementation. The benchmark geometries used in the manuscript are generated from the case definitions reported in the paper.
